# 🚀 Diary of the Future

> 🕰️ Твой персональный AI-ассистент в Telegram для жизни и будущего

---

## ✨ О проекте

**Diary of the Future** - это многофункциональный Telegram-бот, который помогает управлять жизнью, целями и мыслями в одном месте

Он объединяет дневник, напоминания, привычки и AI-анализ в одного удобного ассистента

---

## ⚡ Возможности

* 📝 Личный дневник и заметки
* ⏰ Напоминания и задачи
* 🎯 Цели с дедлайнами
* 🔁 Отслеживание привычек
* 🔍 Поиск по записям и тегам
* 📊 Стрики активности
* 📎 Поддержка вложений
* 📤 Экспорт данных
* 🤖 AI-планирование
* 🧠 AI-анализ дня и целей

---

## 🛠 Стек технологий

* Python
* python-telegram-bot
* SQLite
* OpenAI API

---

## 🚀 Установка

```bash
git clone https://github.com/ICEOF7/diary-of-the-future.git
cd diary-of-the-future
pip install -r requirements.txt
```

---

## ⚙️ Настройка

В файл `.env` добавь:

```env
TOKEN=your_telegram_bot_token
OPENAI_API_KEY=your_api_key
ADMIN_IDS=your_telegram_id
```

---

## ▶️ Запуск

```bash
python app.py
```
